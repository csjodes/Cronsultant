About this terminal-based cron expression editor

What is Cronsultant?

Cronsultant is a terminal-inspired cron expression editor that helps you create, validate, and understand cron expressions with ease. Built with a clean, developer-friendly interface that mimics the terminal experience you know and love.

Features

  • Real-time Validation: Instant feedback on your cron expressions
  • Human-readable Descriptions: See exactly when your cron will run
  • Timezone Support: Preview execution times in different timezones
  • Quick Templates: Common cron patterns ready to use
  • Interactive Challenges: Test your cron knowledge with built-in challenges
  • Copy to Clipboard: Easy copying of validated expressions
  • Dark/Light Themes: Choose between Monokai and Solarized Light themes

How to Use

  1. Enter your cron expression in the input field
  2. Watch as it's validated and described in real-time
  3. Use the field labels below to understand each part
  4. Select a timezone to see when it will run
  5. Copy the expression when you're satisfied
  6. Try the templates for common patterns
  7. Challenge yourself with the Lightning Challenges

Cron Expression Format

Cron expressions consist of five fields separated by spaces:

* * * * *
minute hour day month weekday

Each field can contain:

  • * - Any value
  • 5 - Specific value
  • 1-5 - Range of values
  • 1,3,5 - List of values
  • */5 - Step values (every 5th)

Examples

0 9 * * 1-5 Every weekday at 9:00 AM
*/15 * * * * Every 15 minutes
0 0 1 * * First day of every month at midnight
30 14 * * 2 Every Tuesday at 2:30 PM

Technology

Cronsultant is built with vanilla JavaScript, HTML, and CSS. It uses a custom cron parser for validation and description generation, with support for timezone calculations and upcoming execution previews.
